漢字にふりがな、furigana.js サンプル

漢字混じりの文章をるび付きHTMLに変換する furigana.js。
漢字とかなの対応CSVファイルを用意して、scriptタグを足すだけ、簡単！ (sample1)

<script type="module"> import furigana from "./furigana.mjs"; window.onload = async () => await furigana.addRuby("kana.csv"); </script>

特定のタグ内のみ指定する方法 (sample2)

<script type="module"> import furigana from "./furigana.mjs"; window.onload = async () => { const makeRubyText = await furigana.addRuby("kana.csv"); main.innerHTML = makeRubyText(main.innerHTML); }; </script>
