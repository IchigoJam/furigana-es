漢字にふりがな、furigana-es

特定のタグ内のみ指定する方法

<script type="module"> import furigana from "https://ichigojam.github.io/furigana-es/furigana.mjs"; window.onload = async () => { const makeRubyText = await furigana.addRuby("kana.csv"); main.innerHTML = makeRubyText(main.innerHTML); }; </script>
