漢字にふりがな、furigana.js

漢字混じりの文章をるび付きHTMLに変換する furigana.js。
漢字とかなの対応を書いたCSVファイル(utf-8)を用意して、scriptタグを足すだけ、簡単！ (sample1)

<script type="module"> import furigana from "https://ichigojam.github.io/furigana-js/furigana.mjs"; window.onload = async () => await furigana.addRuby("kana.csv"); </script>
